Case Study - projectConnex

Type

Multipage Application

Tech Stack

  • UX/UI - Bootstrap
  • UX/UI - VueJs
  • API - Laravel
  • Database - MySQL

Overview

This project started out as an idea for a niche social networking platform for developers and later turned into a platform to share and search for open source projects. Building a plat form to bridge the gap between self taught developers and the experience of being in a classroom with other people around your skill level. This project gave me the opportunity to learn the basics of VueJs and Laravel.

Features

  • Login / Sign Up
  • Email Verification
  • Sign Up with 3rd Party (Github)
  • Search Projects
